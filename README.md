#### Реализованная функциональность:

Со стороны пользователя:

*   подача заявки на оформление кредита;
*   расчет ежемесячного платежа, исходя из введенных данных;
*   просмотр статусов поданных заявок;
*   подсказки для получения положительного ответа со стороны системы/сотрудника банка;

Со стороны системы:

*   анализ данных и прогнозирование кредитоспособности клиента на основе его финансового поведения;

Со стороны сотрудника банка:

*   просмотр всех заявок, фильтрация по категориям, статусам заявок, сумме кредита, сроку кредита, поиск по ФИО клиента;
*   одобрение/отклонение заявок, которые система отправляет на ручную проверку;
*   просмотр отдельной заявки + вывод всех данных о заявителе, прошлых заявках и сведений о них;
*   просмотр всех клиентов, фильтрация по году рождения, количеству заявок, категориям и статусам заявок;
*   просмотр информации о клиенте, список всех его заявок;

#### Особенность проекта (киллерфичи):

*   Черный ящик (реестр кредитных сведений);
*   Расчет ежемесячного платежа и его влияния на результат;
*   Различные виды кредитования, влияющие на ставку и результат;
*   Рекомендации по увеличению шанса на получение кредита;
*   Дополнительная ручная проверка на определенных пороговых значениях целевого признака;
*   Админ панель для сотрудников банка с фильтрацией и поиском клиентов/заявок;

#### Стек технологий:

*   python, flask, scikit-learn.
*   javascript, express.js, react.js, react native.
*   mongodb.
